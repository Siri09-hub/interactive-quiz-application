# Quiz Application

A responsive quiz web application built using HTML, CSS, and JavaScript. This project allows users to attempt multiple-choice questions and get instant score results with an interactive interface.

## 🚀 Features
- Multiple-choice questions
- Instant score calculation
- Dynamic question navigation
- User-friendly interface
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
quiz-app/
│── index.html
│── style.css
│── script.js

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Start the quiz
